\subsection{Detailed Description}
\doxyref{POSTable}{p.}{classcma_1_1POSTable} stores the global table of part-\/of-\/speech tags. \doxyref{POSTable}{p.}{classcma_1_1POSTable} stores the global table of part-\/of-\/speech tags, each tag includes a string value and its index code. Note that the string value of part-\/of-\/speech tag is case-\/insensitive. For example, 'Ng' equals to 'ng'.
