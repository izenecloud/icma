\subsection{Detailed Description}
\doxyref{Trainer}{p.}{classmaxent_1_1Trainer} class provides an abstract interface to various training algorithms.

Usually you need not use this class explicitly. \doxyref{MaxentModel::train()}{p.}{classmaxent_1_1MaxentModel_a260e12fb9bbba9ba593819015678313} provides a wrapper for the underline \doxyref{Trainer}{p.}{classmaxent_1_1Trainer} instances.
