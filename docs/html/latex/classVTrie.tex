\subsection{Detailed Description}
\doxyref{VTrie}{p.}{classVTrie} is a variant of the Radix Tree. While the Radix Tree is a space-\/optimized standard trie, the \doxyref{VTrie}{p.}{classVTrie} is a space-\/optimized of the Radix Tree.
